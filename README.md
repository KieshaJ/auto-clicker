### Usage
```bash
gh repo clone KieshaJ/auto-clicker
cd .\auto-clicker\
python -m venv ./venv
source ./venv/bin/activate (if on Mac OS/Linux) | ./venv/bin/activate (if on Windows)
python .\main.py
```
